# Pomodoro Timer

A simple Pomodoro Timer built with React and Vite.

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

## Deploy to GitHub Pages

```bash
npm run deploy
```

This project is configured to deploy to:

`https://yeshacodes.github.io/Pomodoro-Timer.github.io/`

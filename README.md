# 🍅 Pomodoro Timer App

A modern, clean, and interactive **Pomodoro Timer** built using React. This app helps users stay focused, manage tasks, and improve productivity using the Pomodoro technique.

🔗 **Live Demo:** https://yeshacodes.github.io/Pomodoro-Timer.github.io/
📂 **GitHub Repo:** https://github.com/yeshacodes/Pomodoro-Timer.github.io

---

## ✨ Features

* ⏱️ **Pomodoro Timer**

  * Focus, Short Break, and Long Break modes
  * Circular progress timer with clean UI
* 🎯 **Task Management**

  * Add tasks
  * Track what you're working on
* 🌙 **Light & Dark Mode**

  * Smooth theme switching
* 💾 **Persistent Data**

  * Tasks and sessions stored using localStorage
* 🎨 **Modern UI**

  * Clean layout with responsive design
  * Dark mode optimized for better focus

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Styling:** CSS
* **State Management:** React Hooks
* **Deployment:** GitHub Pages

---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/yeshacodes/Pomodoro-Timer.github.io.git
cd Pomodoro-Timer.github.io
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run locally

```bash
npm run dev
```

## 🌐 Deployment

To deploy updates:
